آموزش مرحله به مرحله و خیلی ساده SNI Spoofing

۱. دانلود برنامه Patterniha

آخرین نسخه برنامه رو دانلود کنین (فایل zip یا rar).

https://github.com/patterniha/SNI-Spoofing/releases/tag/v1.0

۲. استخراج فایل‌ها

فایل رو استخراج کنین. داخل پوشه باید فایل Patterniha.exe باشه.

۳. فایل JSON رو آماده کنین

تو همون پوشه‌ای که Patterniha.exe هست، فایل config.json رو ویرایش کنین
بعد این محتوا رو دقیق کپی کنین داخلش:

{
  "LISTEN_HOST": "0.0.0.0",
  "LISTEN_PORT": 40443,
  "CONNECT_IP": "104.19.229.21",
  "CONNECT_PORT": 443,
  "FAKE_SNI": "www.hcaptcha.com",
  "QUEUE_NUM": 1000,
  "HANDSHAKE_TIMEOUT_MS": 2000
}

۴. برنامه رو اجرا کنین

- روی فایل Patterniha.exe راست کلیک کنین

- گزینه Run as administrator رو بزنین (خیلی مهمه!)

۵. کانفیگ V2Ray/Xray/Nekobox/v2rayNG رو تنظیم کنین

تو کانفیگی که داری فقط این دو تا رو عوض کن:

- Address یا آدرس سرور → 127.0.0.1
- Port یا پورت → 40443

بقیه چیزا (UUID، path، TLS، Reality و ...) رو همون قبلی نگه دار.

۶. وصل شو

کانفیگ رو ذخیره کن و بزن Connect. باید وصل بشه.

نکات مهم:

اگه پترنیها ران نشد یه بار از کانفیگ real delay بگیرن استارت میشه
- همیشه Patterniha رو به عنوان مدیر (Run as administrator) اجرا کن.
- اگه ویندوز فایروال پرسید، اجازه دسترسی بده.
- اگه وصل نشد: برنامه رو Stop کن، دوباره Start کن، یا یه config.json جدید با IP دیگه امتحان کن.
