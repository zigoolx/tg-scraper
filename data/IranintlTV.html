با توجه به قطعی‌ها، محدودیت‌ها و هزینه بالای اینترنت در ایران، اگر نمی‌خواهید ویدیوهای کانال ما به‌صورت خودکار برایتان دانلود یا پخش شود، پیشنهاد می‌کنیم دانلود و پخش خودکار ویدیو را در تنظیمات تلگرام خود غیرفعال کنید. با این کار، هنگام بازدید از کانال تلگرام ما، حجم کمتری از اینترنت شما مصرف می‌شود.
برای این‌که ویدیوها در تلگرام اتوماتیک دانلود نشوند:

*در موبایل / آیفون / اندروید

تلگرام را باز کن
برو به Settings / تنظیمات
برو به Data and Storage / داده‌ها و ذخیره‌سازی
در بخش Automatic Media Download / دانلود خودکار رسانه این گزینه‌ها را باز کن:
Using Mobile Data / هنگام استفاده از اینترنت موبایل
Using Wi-Fi / هنگام استفاده از وای‌فای
While Roaming / هنگام رومینگ

داخل هرکدام، گزینه Videos / ویدیوها را خاموش کن.
اگر می‌خواهی هیچ چیزی خودکار دانلود نشود، همه گزینه‌ها مثل عکس، فایل، ویدیو و گیف را هم خاموش کن.

همچنین پایین‌تر، در بخش Auto-play Media / پخش خودکار رسانه، گزینه Videos / ویدیوها را خاموش کن تا ویدیوها حتی خودکار پخش هم نشوند. تلگرام امکان تنظیم دانلود خودکار بر اساس نوع رسانه، نوع چت و حجم فایل را دارد.

*در تلگرام دسکتاپ

برو به Settings / تنظیمات
بعد Advanced / پیشرفته
بخش Automatic Media Download / دانلود خودکار رسانه
برای Private Chats / چت‌های خصوصی، Groups / گروه‌ها و Channels / کانال‌ها گزینه ویدیو را خاموش کن.

برای حالت کم‌مصرف‌تر، علاوه بر خاموش کردن دانلود خودکار، بهتر است هر چند وقت یک‌بار از مسیر Data and Storage > Storage Usage کش تلگرام را هم پاک کنی؛ این کار فایل‌های ذخیره‌شده روی گوشی را حذف می‌کند، نه خود پیام‌ها را.
