آموزش متصل کردن آیفون (iOS) بوسیله برنامه شیروخورشید در اندروید

برای این آموزش اول باید یه گوشی اندروید داشته باشید و بعد طبق مراحل پیش برید:

1- بعد از اتصال در برنامه شیروخورشید، برید توی قسمت Option و More Options، تیک گزینه Share proxy on network رو بزنید و به صفحه اصلی برنامه برگردید

2- الان یه نوشته‌ی سبز رنگ ظاهر شده که دوتا آیپی و پورت بهتون داده
یه آیپی و پورت که HTTP هست و یه آیپی و پورت که SOCKS5 هست

طبق اسکرین شات:
این میشه آیپی که به ما داده: 192.168.43.38 و جلوی علامت (:) هرچی که نوشته باشه میشه پورتی که برنامه بهتون داده (به ما اینجا پورت اچ‌تی‌تی‌پی رو 35339 داده و پورت ساکس رو هم 37009 داده)

3- حالا هاتسپات گوشی اندروید رو روشن کنید و با آیفونتون بهش وصل شید. روی علامت تعجب جلوی اسم وایفای گوشیتون بزنید و آخرین گزینه یعنی Configure Proxy رو روی Manual قرار بدین

4- توی قسمت Server همون آیپی HTTP که برنامه بهتون داده بود و توی قسمت Port هم همون پورتی که برنامه داده بود رو وارد کنید و سیو کنید

5- حالا کل گوشی آیفونتون به وی پی ان وصله و هر برنامه ای رو بخواید میتونید باز کنید اما تلگرام باهاش بالا نمیاد. اگه میخواین تلگرام هم باهاش بالا بیاد وارد تلگرامتون بشید و به قسمت پروکسیاتون برید و یه پروکسی جدید اضافه کنید که پروتوکلش SOCKS5 باشه و توی کادر اولش همون آیپی ساکسی که توی اپ شیروخورشید بود و توی پورتش هم همون پورت ساکس رو وارد کنید و وصل شید✅

نکات:

- اگه فقط میخواین تلگرامتون با آیفون بیاد بالا فقط هاتسپات گوشی اندرویدو روشن کنید و بیاید تلگرام پروکسی ساکس رو بسازید. نیاز به انجام مرحله 3 و 4 نیست اما اگه میخواین کل گوشی از حالت وی پی ان استفاده کنه باید تمام مراحلو انجام بدین
آیپی اچ‌تی‌تی‌پی مختص وارد کردن در تنظیمات آیفون و ساکس مختص به تلگرام هست

- اگه بعد از انجام مراحل وصل نشدید، حتما تنظیماتی که توی قسمت وایفایِ گوشی آیفون انجام داده بودید رو به حالت اولیه برگردونید و گزینه Configure Proxy رو آف کنید تا به حالت اولیه برگرده

1- همه چی مثل آموزش بالاییه یعنی به اپ شیروخورشید در اندروید وصل شید و تیک share proxy on network رو بزنید و گوشیتونو هاتسپات کنید و با ویندوز به وایفای گوشیتون وصل شید

2- توی ویندوزتون سرچ کنید Proxy و وارد تنظیمات پروکسی سیستمتون بشید و روشنش کنید

3- توی قسمت Proxy ip address همون آیپی HTTP که توی اپ شیروخورشید بهتون داده بود رو وارد کنید و پورتش رو هم از همونجا بزنید. حالا سیستمتون هم به وی پی ان وصله ولی تلگرام بالا نمیاد. برای بالا آوردن تلگرام باید توی تلگرام وارد قسمت پروکسیاتون بشید و روی گزینه Use System proxy بذارین تا از پروکسی سیستم که روش ست کرده بودین استفاده کنه و وصل شه
