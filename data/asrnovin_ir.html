🚀 معرفی پروژه Open Source: DDNS Khalil v2

تا حالا خواستید یک سرویس ساب‌دامنه راه‌اندازی کنید، اما نمی‌خواستید کاربران مجبور باشند دامنه شخصی تهیه کنند؟

پروژه DDNS Khalil v2 یک سیستم متن‌باز برای راه‌اندازی سرویس ارائه ساب‌دامنه است که به مدیران اجازه می‌دهد دامنه‌های خود را به سیستم متصل کرده و به کاربران امکان ساخت رکورد DNS از طریق سایت را بدهند.

در این پروژه، مدیر سیستم می‌تواند یک یا چند دامنه را از طریق Zone و Cloudflare API به پنل متصل کند و کاربران پس از ثبت‌نام، قادر خواهند بود برای ساب‌دامنه‌های موجود رکورد ایجاد کنند.

✨ امکانات اصلی پروژه:
🔹 ارائه ساب‌دامنه رایگان به کاربران
🔹 امکان تعریف چندین دامنه توسط مدیر سیستم
🔹 مدیریت کامل DNS با استفاده از Cloudflare API
🔹 ساخت رکوردهای DNS توسط کاربران از طریق پنل
🔹 سیستم ثبت‌نام و مدیریت کاربران
🔹 امکان تعریف پلن‌های رایگان و حرفه‌ای (Pro)
🔹 تبدیل کاربران رایگان به مشتریان پولی با سیستم پلن‌بندی
🔹 اتصال مستقیم خرید پلن به پشتیبانی تلگرام مدیر
🔹 امکان تنظیم آیدی تلگرام مدیر برای دریافت درخواست‌های خرید

روند کار ساده است:
1️⃣ مدیر دامنه‌های موردنظر خود را در سیستم اضافه می‌کند.
2️⃣ کاربر در سایت ثبت‌نام می‌کند.
3️⃣ کاربر می‌تواند از ساب‌دامنه‌های ارائه‌شده یک رکورد رایگان ایجاد کند.
4️⃣ در صورت نیاز به امکانات بیشتر، پلن Pro را خریداری می‌کند و از طریق تلگرام با مدیر سرویس ارتباط می‌گیرد.


این پروژه می‌تواند پایه‌ای برای راه‌اندازی سرویس‌های ساب‌دامنه، DDNS شخصی، سرویس‌های Self-hosted و سیستم‌های ارائه DNS باشد.

📂 سورس پروژه (Open Source):
https://github.com/admin6501/ddns-khalilv2
