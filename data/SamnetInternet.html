🔗 معرفی ابزار chinvat-mx — مالتی‌پلکسر DNS برای پایداری تانل

دوستان، یکی از اعضای کانال یه ابزار ساخته که می‌خوام معرفیش کنم.

🔹 ابزار chinvat-mx چیه؟
یه پروکسی DNS هست که روی VPS ایرانی نصب میشه و ترافیک تانل DNS (مثل DNSTT) رو به جای یه ریزالور، همزمان به چندین ریزالور میفرسته. هر کدوم زودتر جواب بده، برنده‌ست.

🔹 چرا مهمه؟
مشکل اصلی تانل DNS تو ایران اینه که اگه یه ریزالور بمیره یا فیلتر بشه، کل تانل قطع میشه. chinvat-mx این مشکل رو حل می‌کنه:

✅ چندین ریزالور همزمان — اگه یکی بیفته بقیه هستن
✅ هلث‌چک خودکار — هر ۳۰ ثانیه ریزالورهای مرده حذف میشن
✅ ضد شناسایی DPI — ترافیک بین ۵ تا ۱۰ ریزالور پخش میشه و شبیه ترافیک عادی دیتاسنتر به نظر میرسه
✅ سرعت بالاتر — همیشه سریع‌ترین جواب برمی‌گرده

🔹 چطوری کار می‌کنه؟

کاربر ← DNSTT/SlipNet ← chinvat-mxروی VPS ایران ← ریزالورهای ایرانی ← سرور DNSTT خارج ← اینترنت آزاد

هر کوئری DNS که میاد، همزمان به همه ریزالورهای سالم فرستاده میشه. اولین جوابی که برسه برمی‌گرده به کلاینت. بقیه دور ریخته میشن.

🔹 نصب و استفاده
فقط دو تا فایل — پایتون ۳ خالص بدون هیچ وابستگی خارجی. یعنی حتی وسط قطعی اینترنت هم نصب میشه چون نیازی به pip نداره.

📎 گیت‌هاب:
https://github.com/arielesfahani/chinvat-mx/

🙏 تشکر ویژه از سازنده این ابزار که بین ماست. کار ارزشمندیه و مطمئنم به خیلیا کمک می‌کنه. دستت درد نکنه 🌹

🚀 آپدیت Range Scout نسخه 0.1.6

اسکن DNS حالا خیلی شبیه‌تر به SlipNet کار می‌کنه، تست DNSTT داخل خود برنامه انجام میشه و رابط کاربری روی صفحه‌های کوچیک و ویندوز پایدارتر شده.

——————————————

✨ چی عوض شده:

🔹 اسکن DNS به سبک SlipNet بازنویسی شد
۶ تا probe مختلف برای تست سازگاری تانل + امتیازدهی به ریزالورها

🔹 سیستم امتیازدهی اضافه شد
فقط ریزالورهایی که نمره کافی بگیرن میرن مرحله تست DNSTT

🔹حالا DNSTT داخل خود برنامه‌ست
دیگه نیازی به نصب جداگانه dnstt-client نیست

🔹 تنظیمات جدید
—قابلیت E2E URL با مقدار پیش‌فرض generate_204

— و export نتایج scan و DNSTT بهبود پیدا کرد
— دکمه Copy Passed برای کپی سریع ریزالورهای موفق بدون ذخیره فایل

🚀 آپدیت Range Scout نسخه 0.7.0

این نسخه اسکن DNS و مرحله DNSTT را خیلی منعطف‌تر کرده. حالا هم در اسکن می‌تونی بین UDP / TCP / BOTH انتخاب کنی، هم در مرحله DNSTT از UDP / TCP / DOT / DOH استفاده کنی. تنظیمات جدید هم داخل UI و فایل config کامل ذخیره و لود می‌شن.

——————————————

✨ چی عوض شده:

🔹 پروتکل اسکن DNS قابل انتخاب شد
حالا اسکنر فقط UDP نیست و می‌تونی UDP، TCP یا BOTH را برای probeهای سازگاری انتخاب کنی

🔹 مرحله DNSTT حرفه‌ای‌تر شد
برای تست DNSTT حالا UDP، TCP، DOT و DOH پشتیبانی می‌شن

🔹 Resolver Endpoint اضافه شد
برای DNSTT می‌تونی endpoint دلخواه بدی و از placeholderهای {ip} و {port} استفاده کنی

🔹 تنظیمات جدید داخل config ذخیره می‌شن
فیلدهای protocol، transport و resolverURL حالا موقع Save Config نگه داشته می‌شن و دوباره درست لود می‌شن

🔹 اسکن TCP و حالت BOTH به هسته اسکن اضافه شد
نتایج resolverها حالا می‌تونن بر اساس عملکرد واقعی روی UDP و TCP ثبت بشن

🔹 تست‌ها و پایداری بهتر شد
تست‌های scanner، DNSTT و embedded client گسترده‌تر شدن و رفتار transportهای جدید پوشش داده شد

🔹 رابط کاربری و نسخه برنامه آپدیت شد
فیلدهای جدید DNSTT وارد UI شدن و نسخه داخل هدر روی v0.7.0 قرار گرفت

✨ قابلیت‌ها:

🔹 مشاهده DNSها با /dns
با این دستور آخرین DNSهای ثبت‌شده را می‌بینید

🔹 فیلتر بر اساس operator
داخل لیست DNSها می‌تونید operator موردنظرتون را انتخاب کنید و فقط همان نتایج را ببینید

🔹 صفحه‌بندی نتایج
اگر تعداد DNSها زیاد باشد، بات آن‌ها را صفحه‌بندی می‌کند تا راحت‌تر بین نتایج جابه‌جا شوید

🔹 کپی راحت DNSهای هر صفحه
با گزینه Copy All، بات کل DNSهای همان صفحه را در فرمت مناسب برای کپی می‌فرستد

🔹 نمایش مرتب و ساده
نتایج به‌صورت مرتب همراه با operator و زمان ثبت نمایش داده می‌شوند تا پیدا کردن DNS
مناسب راحت‌تر باشد

——————————————

دیدن لیست DNSها:
/dns

از دکمه‌های operator داخل همان پیام استفاده کنید

روی Copy All بزنید

🔗 معرفی کانال White DNS

دوستان عزیز، یه کانال فوق‌العاده کاربردی رو میخوام بهتون معرفی کنم:

کانال 👈 @whitedns

این کانال توسط یکی از دوستان عزیز و قابل اعتمادم مدیریت میشه و تمرکزش روی تکنولوژی DNS Tunnel و ابزارهای دسترسی آزاد به اینترنته.

📌 چی توی این کانال پیدا میکنید:

• کانفیگ‌های آماده DNS Tunnel DNSTT, SlipNet و ...
• لیست DNS Resolver های فعال و به‌روز
• ابزار Range Scout برای پیدا کردن DNS های فعال
• ابزار FindNS برای تست سرتاسری تانل DNS
• ربات @dns_resolvers_bot برای دسترسی سریع به لیست ریزالورها. کار ایشون و دوستای مورد اعتماد خودمون هست.

اگه دنبال راهکارهای DNS Tunnel هستید، این کانال رو از دست ندید. عضو بشید و حمایتش کنید 🙏
