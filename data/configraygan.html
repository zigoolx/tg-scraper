وارد کردن کانفیگ DNSTT در آیفون با HTTP Injector

در این راهنما یاد می‌گیرید چگونه کانفیگ DNSTT در آیفون وارد کنید، این کار معمولاً به دو روش انجام می‌شود:

➖➖➖➖➖➖

☯️ روش اول: وارد کردن کانفیگ آماده (فایل .ehi یا لینک)

این ساده‌ترین و رایج‌ترین روش است.

1️⃣ نصب برنامه

ابتدا اپلیکیشن HTTP Injector را دانلود و نصب کنید.
پیشنهاد می‌شود از آخرین نسخه برنامه استفاده کنید.

➖➖➖➖➖➖

2️⃣ دریافت کانفیگ

کانفیگ معمولاً به یکی از این دو شکل ارائه می‌شود:

• فایل .ehi
• لینک Cloud Config

این فایل یا لینک را از کانال های تلگرام دانلود و در گوشی ذخیره کنید.

➖➖➖➖➖➖

3️⃣ باز کردن برنامه

برنامه HTTP Injector را اجرا کنید.

➖➖➖➖➖➖

4️⃣ ورود به بخش Import

از داخل برنامه:

• روی آیکون ☰ منو (سه خط) بزنید
• گزینه Import Config را انتخاب کنید

➖➖➖➖➖➖

5️⃣ وارد کردن کانفیگ

اگر فایل .ehi دارید:

• گزینه Import from Storage را بزنید
• فایل را از Files / Downloads انتخاب کنید

اگر لینک کلود دارید:

• گزینه Import from URL / Cloud Config را بزنید
• لینک را Paste کنید
• سپس روی Import بزنید

➖➖➖➖➖➖

6️⃣ انتخاب کانفیگ

بعد از وارد شدن کانفیگ:

• کانفیگ در لیست ظاهر می‌شود
• روی آن ضربه بزنید تا فعال شود ✔️

➖➖➖➖➖➖

7️⃣ بررسی فعال بودن DNSTT

در تنظیمات برنامه مطمئن شوید:

• Tunnel Type = DNSTT
• تنظیمات DNS داخل کانفیگ وجود داشته باشد

(معمولاً کانفیگ‌های آماده از قبل تنظیم شده‌اند.)

➖➖➖➖➖➖
8️⃣ اتصال به سرور

حالا:

• دکمه Start / Connect را بزنید
• اولین بار iOS درخواست VPN Permission می‌دهد → گزینه Allow را انتخاب کنید

➖➖➖➖➖➖

9️⃣ اتصال موفق

اگر اتصال برقرار شود:

🟢 وضعیت برنامه Connected می‌شود
🌍 آی‌پی اینترنت شما تغییر می‌کند

➖➖➖➖➖➖

☯️ روش دوم: ساخت دستی کانفیگ DNSTT + SSH

اگر فقط مشخصات سرور را دارید، باید کانفیگ را دستی بسازید.

➖➖➖➖➖➖

1️⃣ باز کردن برنامه

اپلیکیشن HTTP Injector را اجرا کنید.

➖➖➖➖➖➖

2️⃣ ورود به بخش SSH

به قسمت SSH Tunnel بروید و اطلاعات زیر را وارد کنید:

• Host / IP
• Port (معمولاً 22 یا 443)
• Username
• Password

➖➖➖➖➖➖

3️⃣ فعال کردن DNSTT

به بخش DNS یا DNSTT Settings بروید و گزینه DNSTT / DNS Tunnel را فعال کنید.

➖➖➖➖➖➖

4️⃣ وارد کردن اطلاعات DNSTT

فیلدهای زیر را دقیق وارد کنید:

• DNSTT Server / Nameserver
مثال:
dns.example.com:443

• DNSTT Public Key
کلید عمومی سرور که معمولاً طولانی است.

➖➖➖➖➖➖

5️⃣ تنظیم Resolver

معمولاً یکی از DNS های زیر استفاده می‌شود:

• 8.8.8.8
• 1.1.1.1

➖➖➖➖➖➖
6️⃣ ذخیره تنظیمات

در صورت نیاز:

• روی Save بزنید
• سپس به صفحه اصلی برگردید

➖➖➖➖➖➖

7️⃣اتصال

اکنون دکمه Start را بزنید و چند ثانیه صبر کنید تا اتصال برقرار شود.

➖➖➖➖➖➖

⚠️ نکات مهم برای کاربران آیفون

🔹 اگر اتصال ناپایدار بود، تنظیمات Send Buffer / Receive Buffer را روی 4096 یا 8192 تست کنید.

🔹 حالت Low Power Mode آیفون را خاموش کنید.

🔹 در تنظیمات گوشی بروید به:
Settings → HTTP Injector → Background App Refresh
را فعال کنید.

🔹 بعضی کانفیگ‌ها نسخه قدیمی هستند و ممکن است کار نکنند؛ بهتر است از کانفیگ‌های جدید استفاده کنید.

➖➖➖➖➖➖

✅ حالا می‌توانید از اینترنت آزاد با DNSTT استفاده کنید.

داخل اپلیکیشن، بر روی علامت ⚙️ بزنید و تغییرات زیر رو انجام بدید.

١. گزینه‌ی Auto-reconnect رو روشن کنید.

٢. گزینه‌ی Disable Quick رو روشن کنید.

٣. تغییر مقدار VPN MTU

بر روی گزینه VPN MTU بزنید، روی Custom بذارید و مقدار 512 رو تایپ کنید و Apply رو بزنید.

۴. گزینه‌ی Enable geo-bypass رو روشن کنید و Country رو روی Iran بگذارید.

۵. گزینه‌ی Enable split tunneling رو روشن کنید، Mode رو روی VPN بذارید و در قسمت Select apps، اپلیکیشن‌های مورد نظر رو انتخاب کنید.

۶. گزینه‌ی Compression رو روشن کنید.

٧. گزینه‌ی Cipher رو روی AES-128-GCM بگذارید.

٨. گزینه‌ی Max channels رو روی Auto بگذارید.

٩. لینک‌های کانفیگ که در مرحله‌ی راه‌اندازی سرور کپی کردید رو با زدن علامت + و انتخاب Import وارد برنامه کنید.

١٠. برای اتصال، نیاز به یک DNS Resolver دارید، که با استفاده از قابلیت اسکن داخلی نرم‌افزار، یا جست‌وجو در کانال‌ها و گروه‌ها، می‌تونید به دست بیارید.
